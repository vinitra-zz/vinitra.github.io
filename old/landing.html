Projects

A collection of analysis, research, and exploration projects
I've started over the last few years.

Research

I joined the Berkeley Institute of Data Science in the Fall 2015 semester, and continued on as a team lead in the Spring 2016, Summer 2016, Fall 2016, and Spring 2017 Semesters. My initial work was with the Curriculum team to develop data science modules for classes on campus using NLP, statistical analysis, and interactive visualization techniques in iPython notebooks. There was also some ethnographic work done on the new Data 8 pilot class by conducting focus groups and creating surveys.

Mapping Projects

The next semester, I continued as head of the Mapping Team, a team dedicated to scraping and analyzing Berkeley relevant data sets in a way that is useful to the Data Science Planning initiative. One of our major projects, Course Pathways involves scraping the course catalog and using NLP techniques to identify data centric courses, then developing a classification algorithm to group these classes into clusters. We focused on mapping prerequisite relationships, identifying type of class, developing a filterable system (in the works), and creating interactive D3.js visualizations by cluster.

Filterable Course Classification D3.js Course Cluster Visualizations

Another major project dealt with Research Grant Mapping, which consisted of scraping the Vice Chancellor of Research's database as well as the Sponsored Projects Office data set to get a listing of over 15 years of Berkeley Research Grant Data. The exploratory data analysis on the structure of Berkeley Research Funding was done in an iPython notebook.

Research Grant Analysis

Translations of Onegin

This project was a computational text analysis approach to comparing different translations of the internationally acclaimed Alexander Pushkin's Eugene Onegin. Tools and techniques used include the Natural Language Tool Kit, mapping frequencies to n-dimensional vector spaces, using cosine similarity as a comparison metric, and multidimensional scaling.

  • Computational Analysis of Pushkin

Election Prediction Model

Using census data to do an exploratory data analysis and a time series analysis of political slant by district in the elections from 2008 to 2016.

  • Election Prediction Model

Bechdel Test

The Bechdel Test evaluates films on the basis of gender disparity. This quick CalHacks project was a computational exploration into the Bechdel Test dataset to understand the fundamental differences between films that pass and fail.

  • Bechdel Analysis
